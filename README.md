# d5-modules

Code exemple pour la création de modules Divi
Repository GitHub : https://github.com/chergreg/d5-modules.git 
Auteur : Gregory Borelli


Ce code est basé sur le repo git Elegant Theme Officiel
https://github.com/elegantthemes/d5-extension-example-modules

Seul le module le plus simple Static-Module a été gardé.
Correction du code pour le rendre compatible avec Divi 5 beta


## Build et Installation
npm run build
npm run install

Nom du plgin : d5-demo-modules.zip
Ensuite installer le plugin dans WordPress comme n'importe quel plugin externe.
